คำถามที่พบบ่อย

ฟรีจริงไหม? มีโฆษณาหรือเปล่า?

ฟรี 100% ไม่มีโฆษณา ไม่มีค่าใช้จ่าย และเป็นโอเพนซอร์ส (GPL-3.0) — ดูโค้ดทั้งหมดได้บน GitHub

ทำไมต้องถอนตัวต้นฉบับก่อน?

เพราะเซ็นด้วยลายเซ็นคนละตัวกับต้นฉบับ Android เลยไม่ให้อัปทับกันโดยตรง ถอนของเดิมออกครั้งเดียวแล้วลงตัวนี้ หลังจากนั้นอัปเดตเวอร์ชันใหม่ทับได้ตลอด

ต้องเปิดแอพค้างไว้ตลอดไหม?

ระบบทำงานเป็น service เบื้องหลัง แนะนำให้ปิด Battery Optimization ให้แอพ เพื่อไม่ให้ระบบฆ่า service ตอนอยู่เบื้องหลัง

ปลอดภัยไหม? ใช้ token ยังไง?

Kizzy ใช้ user token เพื่อแสดง Rich Presence ซึ่งเป็นการใช้งานนอกเหนือข้อกำหนดปกติของ Discord (เขตสีเทา) โปรดใช้อย่างเข้าใจความเสี่ยง token ถูกเก็บไว้ในเครื่องคุณเท่านั้น ไม่ส่งไปไหน

เจอบั๊กหรืออยากเสนอฟีเจอร์?

เปิด issue ได้ที่หน้า GitHub Issues ของเรา
